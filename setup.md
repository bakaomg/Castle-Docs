# Castle Docs - V0.2.6

## 下载主题
首先前往 https://github.com/ohmyga233/castle-Typecho-Theme/releases 下载主题。

## 安装主题
将压缩包解压到 `/usr/themes/` 并将文件夹名称 `castle-Typecho-Theme` 改为 `Castle`。

!> 友链功能需安装 [links](http://www.imhan.com/archives/typecho_links_20141214/) 才能正常使用。
